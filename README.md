# LinkedIn Post Analytics

A beautiful analytics dashboard for your LinkedIn posts, branded with Integral Recruiting Design's aesthetic.

## Features

- Upload LinkedIn export data (CSV or Excel)
- View engagement metrics: impressions, likes, comments, reposts
- Calculate engagement rate automatically
- Sort by any metric to find top performers
- Click through to view posts on LinkedIn
- Clean, gradient-styled interface

## How to Deploy

### Option 1: GitHub Pages (Easiest)

1. Create a new repository on GitHub
2. Upload `index.html` to the repository
3. Go to Settings > Pages
4. Select "Deploy from a branch" and choose "main" branch
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Cloudflare Pages

1. Create a new repository on GitHub with `index.html`
2. Go to Cloudflare Dashboard > Pages
3. Click "Create a project" > "Connect to Git"
4. Select your repository
5. Leave build settings empty (it's a static HTML file)
6. Click "Save and Deploy"

### Option 3: Netlify

1. Drag and drop the `index.html` file into Netlify's deploy interface
2. Or connect your GitHub repo for automatic deployments

## How to Use

1. Export your LinkedIn post data
2. Upload the CSV or Excel file
3. Explore your metrics and find your top-performing content!
